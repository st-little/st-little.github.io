Ubuntu に Time Machine を構築する

Published on 2024/02/23
Tags mac timemachine

概要

Ubuntu に netatalk を使用して Time Machine を構築します。

環境

  • Ubuntu 20.04 LTS
  • netatalk 3.1.12

Time Machine の構築

  1. Time Machine のユーザーを作成します。

    sudo adduser timemachine

  2. バックアップデータを保存するフォルダを作成します。

    mkdir /home/timemachine/TMBackup
chown timemachine:timemachine /home/timemachine/TMBackup

  3. netatalk をインストールします。

    sudo apt install netatalk
netatalk -V

  4. /etc/netatalk/afp.conf を変更して netatalk の設定を行います。

    次のコメントアウトを削除して値を変更します。

    • [My Time Machine Volume]: Mac で表示される名前です。今回は [Proxmox Time Machine] に変更します。
      • path: バックアップデータを保存するフォルダのパスです。今回は /home/timemachine/TMBackup に変更します。
      • time yes で machine: Time Machine を有効にします。
    ;
; Netatalk 3.x configuration file
;

[Global]
; Global server settings

; [Homes]
; basedir regex = /xxxx

; [My AFP Volume]
; path = /path/to/volume

[Proxmox Time Machine]
path = /home/timemachine/TMBackup
time machine = yes

  5. netatalk を再起動します。

    sudo service netatalk restart

  6. Mac の Time Machine から Proxmox Time Machine が選択できることを確認します。
