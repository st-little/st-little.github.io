Windwos に Minecraft Forge 14 をインストールする

December 29, 2022 - game minecraft forge java edition

概要

Windwos に Minecraft Forge 14 をインストールします。

環境

  • Windows 11 Home

    • バージョン 22H2

  • Minecraft

    • Java Edition
    • バージョン 1.12.2

  • Minecraft Forge

    • バージョン 14.23.5.2859

手順

準備

すでに Java がインストール済みの場合は不要です。

  1. Java の ダウンロードページ を開いて、「Javaのダウンロード」ボタンを選択します。

  2. ダウンロードされた「JavaSetup8u351.exe」を実行して Java をインストールします。

Minecraft Forge のインストール

  1. Minecraft 1.12.2 用の ダウンロードページ を開きます。

  2. Download Recommended の Installer を選択します。

  3. 広告画面の 「SKIP」ボタンを選択してインストーラーをダウンロードします。

  4. ダウンロードしたインストーラー「forge-1.12.2-14.23.5.2859-installer.jar」を実行してインストールを開始します。

  5. インストールが完了したら「OK」ボタンを選択します。

起動構成の作成

  1. Minecraft Launcher を起動して「MINECRAFT:Java Edition」を選択します。

  2. 「Installations」タブを選択し「新規作成」ボタンを選択します。

  3. 「名前」に自分が分かりやすい任意の名前を入力します。

  4. 「バージョン」は release 1.12.2-forge-14.23.5.2859 を選択します。

  5. 「ゲームディレクトリ」は「参照」ボタンを選択して任意のフォルダーを選択します。

  6. 「作成」ボタンを選択して起動構成を作成します。

起動の確認

  1. 作成した起動構成の「プレイ」ボタンを選択します。

  2. 警告ダイアログが表示されるので「プレイ」ボタンを選択して Minecraft を起動します。

  3. 無事に Minecraft が起動したら「Quit Game」ボタンを選択して Minecraft を終了します。
