Minecraft Java Edition をコントローラーでプレイする

January 08, 2023 - game minecraft forge java edition mod

概要

Minecraft に Mod (Controllable) を入れてコントローラーでプレイできるようにします。

Forge が未インストールの場合は、先に Forge をインストールしてください。

(Windwos に Minecraft Forge 14 をインストールする)

環境

  • コントローラー

    • Xbox ワイヤレス コントローラー + ワイヤレス アダプタ for Windows 10 1VA-00005

  • Minecraft

    • Java Edition
    • バージョン 1.12.2

  • Minecraft Forge

    • バージョン 14.23.5.2859

  • Controllable

    • バージョン 0.11.2

手順

Controllable のダウンロード

  1. Controllable の Main File を開きます。

  2. Game Version 1.12.2 の Controllable 0.11.2 にあるダウンロードボタンを選択し、 controllable-0.11.2-1.12.2.jar をダウンロードます。

Controllable のインストール

  1. Minecraft Launcher を起動して「MINECRAFT:Java Edition」を選択します。

  2. 「Installations」タブを選択します。

  3. Controllable をインストールする起動構成のフォルダを開きます。

  4. 「mods」フォルダの中に Controllable の controllable-0.11.2-1.12.2.jar ファイルを置きます。

Controllable の確認

  1. コントローラをパソコンに接続します。

  2. Controllable をインストールした起動構成の「プレイ」ボタンを選択します。。

  3. Minecraft が起動したら「Mods」を選択します。

  4. インストール済みの Mod 一覧に Controllable が表示されることを確認します。
