Minecraft に Galacticraft をインストールする

December 30, 2022 - game minecraft forge java edition mod galacticraft

概要

Minecraft に Galacticraft をインストールします。

Minecraft Forge が未インストールの場合は先に Minecraft Forge をインストールしてください。

環境

  • Windows 11 Home

    • バージョン 22H2

  • Minecraft

    • Java Edition
    • バージョン 1.12.2

  • Minecraft Forge

    • バージョン 14.23.5.2859

  • Galacticraft

    • バージョン 4

手順

Galacticraft のダウンロード

  1. Galacticraft の ダウンロードサイト を開きます。

  2. 「Minecraft Version」で「1.12.2」を選択します。

  3. jar ファイルを全てダウンロードします。

Galacticraft のインストール

  1. Minecraft Launcher を起動して「MINECRAFT:Java Edition」を選択します。

  2. 「Installations」タブを選択します。

  3. Galacticraft をインストールする起動構成のフォルダを開きます。

  4. 「mods」フォルダの中に Galacticraft の jar ファイルを置きます。

Galacticraft の起動

  1. Galacticraft をインストールした起動構成の「プレイ」ボタンを選択します。。

  2. Minecraft が起動したら「Mods」を選択します。

  3. インストール済みの Mod 一覧に Galacticraft が表示されることを確認します。
