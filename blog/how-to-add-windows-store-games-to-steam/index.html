Windows Store のゲームを Steam に追加する方法

Published on 2024/01/10
Tags game minecraft steam uwphook

概要

UWPHook を使用して Windows Store のゲームを Steam に追加します。
この記事では Minecraft Launcher を Steam に追加します。

前提条件

  • Steam がインストールされている必要があります。

環境

  • OS: Windows 11 Home 23H2
  • UWPHook: バージョン 2.13.0
  • Steam: バージョン 1702079146
  • Minecraft Launcher: バージョン v.2.15.23-1.7.2

手順

UWPHook のダウンロード

  1. UWPHook の Releases を開きます。

  2. 最新バージョンの　UWPHook.exe を押下してインストーラーをダウンロードします。

UWPHook のインストール

  1. ダウンロードした UWPHook.exe をダブルクリックしてインストーラーを起動します。

  2. Next > ボタンを押下してインストーラーを開始します。

  3. I Agree ボタンを押下してライセンスに同意します。

  4. インストールするフォルダを選択し Next > ボタンを押下します。

  5. Install ボタンを押下してインストールを開始します。

  6. インストールが完了したら Finish ボタンを押下してインストーラーを終了します。

ゲームを Steam に追加

  1. UWPHook を起動します。

  2. SteamGridDB からゲーム画像を自動で読み込むか問われるので、今回は いいえ(N) ボタンを押下します。

  3. ゲームの一覧が表示されます。

  4. Steam に追加するゲームにチェックを入れます。今回は Minecraft Launcher にチェックを入れます。

  5. Export selected apps to Steam ボタンを押下します。

  6. Steam に追加が完了したことを知らせるダイアログが表示されるので OK ボタンを押下します。

  7. Steam のライブラリを開いてゲームが追加されていることを確認します。
