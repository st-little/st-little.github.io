Minecraft Bedrock Edition のサーバーでコマンドを実行する

January 09, 2023 - game minecraft bedrock edition

概要

Minecraft Bedrock Edition のサーバーでコマンドを実行します。

コマンドを実行することでゲームモードの変更や座標を表示することができます。

実行できるコマンドは List of Commands を参照してください。

手順

Docker でサーバーを起動している想定です。

  1. サーバーの CONTAINER ID 確認します。

    docker ps

  2. サーバーのコンテナにアタッチします。

    docker attach {CONTAINER ID}

  3. 任意のコマンドを実行します。試しに gamerule の showcoordinates を true に変更して座標を表示します。

    gamerule showcoordinates true

  4. Minecraft のクライアント側で座標が表示されていることを確認します。

  5. キー「Ctrl-P Ctrl-Q」を入力して、アタッチしたコンテナからデタッチします。
