Minecraft のサーバーに Cocricot をインストールする

Published on 2024/01/08
Tags game minecraft forge java edition mod cocricot

概要

Minecraft Java Edition のサーバーに Cocricot をインストールします。
サーバーに Cocricot をインストールすると、クライアントは Cocricot をインストールする必要はありません。

前提条件

  • 既に Minecraft Java Edition のサーバーが構築されている必要があります。

    • サーバーを構築する方法は こちら を参照してください。

  • サーバーとクライアントのマインクラフトのバージョンは一致している必要があります。

環境

サーバー

  • Minecraft Java Edition: バージョン 1.20.1

  • Forge: バージョン 47.2

  • Cocricot: バージョン 1.20.1

クライアント

  • Minecraft Java Edition: バージョン 1.20.1

手順

リソースパックのダウンロード

公式サイトのダウンロードページからリソースパックをダウンロードします。

  1. リソースパックの ダウンロードページ を開きます。

  2. 1.20.1 Java Edition の Resourcepack を押下します。

  3. 利用規約に同意できる場合は 利用規約に同意してダウンロード を押下します。

リソースパックのアップロード

リソースパックをクラウドストレージサービスにアップロードします。
今回は MCPacks にアップロードします。

  1. MCPacks を開きます。

  2. ファイルを選択 ボタンを押下しリソースパックを選択します。

  3. Upload ボタンを押下してリソースパックをアップロードします。

  4. Download URL, SHA-1 Hash をコピーします。

サーバーの設定

サーバーのコンテナ定義 docker-compose.yml を設定します。
詳細は Custom resource pack を参照してください。

  1. environment の RESOURCE_PACK に Download URL を設定します。

    RESOURCE_PACK: "https://download.mc-packs.net/pack/812efe7330936ebd1fb3d147deb977616077b3a3.zip"

  2. environment の RESOURCE_PACK_SHA1 に SHA-1 Hash を設定します。

    RESOURCE_PACK_SHA1: "812efe7330936ebd1fb3d147deb977616077b3a3"

  3. environment の RESOURCE_PACK_ENFORCE に TRUE を設定します。

    RESOURCE_PACK_ENFORCE: "TRUE"

  4. サーバーを起動します。
